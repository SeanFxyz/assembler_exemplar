# Not Gate

## Description

The "Not" gate takes one input and outputs the opposite (1 to 0, or 0 to 1).

## Operations

```
Inputs: in
Outputs: out
Function: *if* in=0 *then* out=1 *else* out=0
```
* Adapted from *The Elements of Computing Systems* Pg. 19

## Additional Information

Boolean Logic Table:

```
in  |   out
----+---------
0   |   1
1   |   0
```
